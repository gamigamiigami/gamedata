# 帰宅後やること（メール機能の仕上げ・2ステップ）

コード側は全て実装・デプロイ済み。残りは**あなたのダッシュボード操作だけ**（コピペで完了）。
※どちらも私（AI）はアカウント権限が無いため代行できません。

---

## ステップ1：EmailJS の「Contact Us」テンプレを汎用化する

先生（申請者）への「受付メール」「学校コードのメール」は、この1つのテンプレで送ります。

1. emailjs.com → **Email Templates** → **「Contact Us」(template_6c3mcbm)** を開く
2. 各欄を以下に設定（変数は二重波カッコ）：
   - **To Email**：`{{to_email}}`
   - **From Name**：`IOgames`
   - **Subject**：`{{subject}}`
   - **Content（本文）**：`{{message}}`
3. **Save**

> これだけ。テンプレIDは変えない（`template_6c3mcbm` のまま）。
> 「Auto-Reply」(template_gp9y1ai) は禁止ワード通知で使用中なので触らない。

### テスト
ライブの `/teachers/register/` でテスト申請 → 入力した**先生のアドレス宛**に
「申請を受け付けました」メールが届けば成功。

---

## ステップ2：Supabase の通知メールSQLを再実行（承認ボタン付きに更新）

管理者（あなた）への通知メールに「管理画面を開いて承認する」ボタンを追加した版。

1. `supabase/notify-email.sql` の中身をコピー
2. **`YOUR_RESEND_KEY`** を自分のResendキーに置き換える
3. Supabase SQL Editor に貼って **RUN**（`create or replace` なので上書き）

### テスト
`/teachers/register/` でテスト申請 → あなたのGmailに届く通知メールに
黄色い「管理画面を開いて承認する」ボタンがあれば成功。

---

## 全体の流れ（完成後）

```
先生が /teachers/register/ で申請
   ├─ あなた宛：Resend通知メール（承認ボタン付き）        ← ステップ2
   └─ 先生宛 ：EmailJS受付メール「受け付けました」        ← ステップ1

あなたが /admin/ でログイン → 「承認」ボタン
   └─ 先生宛 ：EmailJS「学校コードは ◯◯ です」を自動送信   ← ステップ1のテンプレ流用
```

- 先生への送信は EmailJS（あなたのGmail発）＝ドメイン不要・無料月200通
- あなたへの通知は Resend（pg_net トリガー）
- 将来 独自ドメインを取れば Resend で全部統一も可能（今は不要）
